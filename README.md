# MLE_course_project_1: Приложение Kolibri

Сервис по классификации птиц по изображению для любителей птиц.

## Функционал:

* ### Авторизация/регистрация:

Со стартовой страницы пользователь переходит на форму, где вводит логин и пароль.

* ### Личный кабинет:

Просмотр баланса, пополнение баланса.

* ### Работа с моделью:

На странице личного кабинета пользватель загружает изображение с птицей и получает ответ через несколько секунд, к какому виду она относится.

* ### Просмотр истории предсказаний и транзакций:

Реализована возможность перейти на страницы с историей предсказаний и тарнзакций.

* ### Log Out:

Пользователь может разлогиниться.
